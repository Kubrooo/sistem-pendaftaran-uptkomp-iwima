# Contributing

Terima kasih ingin berkontribusi! Langkah cepat:

1. Fork repo ini
2. Buat branch fitur: `git checkout -b feat/nama-fitur`
3. Jalankan linter/format jika perlu dan tambahkan test jika menambahkan logic
4. Buat PR ke branch `main` dengan deskripsi perubahan

Style
- JavaScript/Node: ikuti konfigurasi ESLint (jika ditambahkan) dan gunakan format konsisten.

Running locally
- Ikuti langkah di README untuk menjalankan server dan client.
